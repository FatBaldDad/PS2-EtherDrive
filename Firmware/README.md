# Firmware README

## Introduction

This folder contains firmware-related documentation, release notes, recovery information, and future binary releases for the PS2-EtherDrive project.

The project is currently in early development and firmware architecture is still under active evaluation.

---

# Firmware Goals

PS2-EtherDrive firmware is intended to provide:

- Embedded networking functionality
- SMB support
- UDPBD support
- WiFi bridge capability
- Internal storage management
- Recovery and update support

# OpenWrt

The project is currently planned around an OpenWrt-based firmware platform.

OpenWrt was selected because it provides:

- Lightweight embedded operation
- Flexible networking support
- Existing package ecosystem
- MIPS platform compatibility
- USB storage support

The exact firmware architecture may evolve significantly during development.
