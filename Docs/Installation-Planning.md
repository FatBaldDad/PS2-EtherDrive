# Installation Planning

## Introduction

PS2-EtherDrive is being designed with internal installation and OEM-style integration as major project goals.

The intention is to create a networking and storage platform that fits cleanly inside supported PlayStation 2 Slim consoles while minimizing visible external modification.

This document tracks installation concepts, fitment planning, and integration goals.

---

# Installation Philosophy

## Internal Integration

The project is intended to be fully internal wherever possible.

Goals include:

- No external router hardware
- No external bridge devices
- Minimal external wiring
- Minimal shell modification
- OEM-style appearance

---

# Target Console Revisions

Initial development is primarily focused on later PlayStation 2 Slim models.

These systems are attractive targets because they lack traditional internal HDD solutions while still supporting network loading workflows.

Compatibility testing will expand as development progresses.

---

# Internal Space Planning

## Available Installation Areas

Several internal installation areas are currently being evaluated.

Areas under consideration include:

- Beside the cooling fan
- Alongside the optical drive area
- Under modified RF shielding
- Behind controller ports
- Custom mounting plate locations

Final placement may vary depending on console revision.

---

# RF Shield Integration

## Thermal and Mechanical Use

The RF shielding may serve multiple purposes:

- Structural mounting support
- Thermal transfer surface
- Mechanical alignment
- PCB support

Current design research includes:

- Copper thermal transfer plates
- Shield-mounted PCB integration
- Low-profile component placement

---

# Ethernet Port Integration

## Original Ethernet Jack

One of the primary design goals is retaining use of the original Ethernet jack.

Current concepts include:

- Relocating Ethernet transformers
- Flex PCB routing
- Internal passthrough routing
- Combined wired and wireless operation

The final routing implementation is still under development.

---

# Storage Integration

## Internal Storage Devices

The project is intended to support internal storage devices such as:

- USB flash drives
- USB SSDs
- microSD storage

Areas being evaluated include:

- Storage mounting
- Cable routing
- Power delivery
- Thermal behavior
- Accessibility

---

# Flex PCB Planning

## Flexible Interconnects

Flex PCBs may be used for:

- Ethernet relocation
- Tight-space routing
- Cleaner installations
- Reduced installation complexity

This remains an active development area.

---

# Cooling and Airflow

## Thermal Preservation

Preserving PS2 cooling performance is considered important.

Current considerations include:

- Avoiding airflow blockage
- Limiting thermal buildup
- Coupling heat into shielding
- Maintaining fan effectiveness

Thermal testing will be a major part of development.

---

# Installation Difficulty Goals

The long-term goal is reducing installation complexity wherever possible.

Areas being considered include:

- Reduced wire count
- Simplified mounting
- Connector-based installation
- Preconfigured firmware
- Repeatable assembly methods

---

# Prototype Installation Process

Current installation workflows are expected to evolve significantly during development.

Prototype installations may initially require:

- RF shield modification
- Internal wiring
- Soldering
- Custom routing
- Manual firmware setup

Long-term goals may differ substantially from early prototype methods.

---

# Future Expansion Possibilities

Possible future additions may include:

- Simplified install kits
- Preassembled modules
- Additional expansion hardware
- Combined mod integration
- Enhanced internal mounting systems
