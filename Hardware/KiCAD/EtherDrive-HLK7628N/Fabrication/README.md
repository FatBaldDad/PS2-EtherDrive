# Fabrication Folder Structure

This folder organizes manufacturing exports for prototype and production handoff.

## OSH Park

- `OSH-Park/Gerbers/` — Gerber outputs
- `OSH-Park/Drill/` — drill files
- `OSH-Park/Assembly/` — assembly references (PDFs, XY, notes)

## Generic Outputs

- `Outputs/` — alternate fab outputs and archived release bundles
