# Hardware README

## Introduction

This folder contains hardware-related documentation, prototype planning, PCB renders, mechanical concepts, and installation research for the PS2-EtherDrive project.

The project is currently in early development and hardware architecture is still evolving.

- Internal networking functionality
- Internal storage support
- Ethernet passthrough capability
- WiFi bridge support

# HLK-7628N Platform

The current hardware platform centers around the HLK-7628N embedded router module.

Reasons for selection include:

- Compact size
- OpenWrt support
- Integrated networking capability
- USB support
- Low power requirements

The exact hardware architecture may continue evolving during development.

---

# Ethernet Passthrough Concept

One of the major project goals is preserving functionality of the original PS2 Ethernet jack.

Current design concepts involve:

- Ethernet transformer relocation
- Internal passthrough routing
- Combined wired and wireless functionality

This remains an active area of hardware research.

---

# Thermal Design

Thermal management is considered a major development priority.

Areas currently under evaluation include:

- RF shield heat transfer
- Passive cooling strategies
- Copper thermal plates
- Internal airflow preservation
- Heat isolation techniques

---

# Mechanical Design Goals

The project emphasizes:

- Internal installation
