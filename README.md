# PS2-EtherDrive

**PS2-EtherDrive** is an **FBD / Fat Bald Dad** project focused on building a compact PlayStation 2 networking and storage platform for PS2 Slim consoles.

The goal is to create a cleaner, more integrated way for compatible PS2 systems to use network-based game loading, storage sharing, Wi-Fi bridging, and PS2-focused setup tools without relying on a bulky external network setup.

This project is being developed as a future FBD kit, product, service install, or documented platform for the PlayStation 2 modding community.  Here is where it all started https://bitbuilt.net/forums/threads/ps2-ultra-slim.5077/post-73650

> [!IMPORTANT]
> PS2-EtherDrive is currently in private development.
>
> This public repository exists to document the project name, public goals, development status, and ownership of the FBD PS2-EtherDrive project.
>
> This is **not** an open-source hardware, firmware, or manufacturing release at this time.
>
> Detailed firmware, PCB files, manufacturing files, installation methods, source code, electrical design files, and internal development notes are intentionally private until the project is further tested and ready for release.

---

## Project Purpose

The PlayStation 2 has several ways to load games and homebrew through network and storage devices, but many setups require extra hardware, external adapters, multiple cables, or user configuration that can be confusing for casual users.

PS2-EtherDrive is intended to explore a more integrated approach.

The long-term goal is to create a PS2-focused network/storage platform that feels like it belongs with the console and is easier to configure, install, and support.

---

## What PS2-EtherDrive Is Trying to Do

PS2-EtherDrive is intended to explore:

- Internal network storage support for compatible PS2 Slim consoles
- PS2-focused network loading workflows
- Simple setup and configuration for users
- Clean integration with the console shell
- Wi-Fi bridge-style functionality where useful
- Retaining useful wired network functionality where possible
- A future web-based setup interface designed around PS2 use
- Better documentation for PS2 networking and storage setups
- A possible future FBD kit, installation service, or documented build platform

The public repository will grow as the project matures, but the working development files are kept private for now.
